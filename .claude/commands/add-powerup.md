Ajoute un nouveau power-up "$ARGUMENTS" au jeu.

1. Crée `src/game/entities/PowerUp.ts` si inexistant
2. Ajoute le type et la couleur dans types.ts
3. Implémente l'effet sur le Player
4. Ajoute le spawn dans GameEngine
5. Ajoute la durée/timer si temporaire

Power-ups suggérés:
- Shield: invincibilité temporaire
- SpreadShot: tir multiple
- SpeedBoost: vitesse augmentée
- Bomb: détruit tous les ennemis proches
